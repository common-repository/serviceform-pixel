User data is managed as per the terms and conditions at (https://serviceform.com/terms)
== Description ==
Serviceform provides software and tools for global clients and customers, particularly chatbot and form builder software. 
== Frequently Asked Questions ==
= Where can I get the pixel ID? =
You have to sign-up at serviceform.com and enter your website details in order to get the Pixel ID. It will be displayed once you click continue after entering the website URL.
If you need our help, please visit (https://serviceform.com/contact)
